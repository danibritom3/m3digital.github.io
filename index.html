Política de Privacidade

Última atualização: 22 de julho de 2025

Esta Política de Privacidade descreve como este aplicativo utiliza a API da Meta (Facebook) para acessar e processar dados de contas de anúncios, exclusivamente com a finalidade de geração de relatórios internos de desempenho publicitário.

1. Coleta de Dados

Este aplicativo acessa, por meio da API da Meta, apenas dados relacionados ao desempenho de campanhas publicitárias, incluindo:

  • Nome da conta publicitária
  • Nome das campanhas
  • Impressões
  • Cliques
  • Alcance
  • Frequência
  • Gasto (spend)
  • Métricas de custo (CPC, CPM, CTR)
  • Ações e conversões (via action_type)

Importante: Este aplicativo não coleta, armazena ou acessa informações pessoais identificáveis de usuários da Meta ou de terceiros.

2. Uso dos Dados

Os dados coletados são utilizados exclusivamente para fins internos, como visualização de desempenho de campanhas, geração de relatórios e tomada de decisões estratégicas.

Não utilizamos os dados para marketing, remarketing ou criação de perfis de usuários.

3. Armazenamento e Segurança

Os dados podem ser armazenados temporariamente em sistemas como Make.com ou Google Sheets. Utilizamos práticas de segurança da informação, como:

  • Transmissão por HTTPS
  • Armazenamento em ambientes controlados
  • Acesso restrito com autenticação

4. Compartilhamento de Dados

Este aplicativo não compartilha dados com terceiros. O acesso é restrito à equipe autorizada.

5. Direitos dos Titulares dos Dados

Você pode solicitar acesso, correção ou exclusão de dados. Envie um e-mail para o contato abaixo. Responderemos em até 30 dias corridos.

6. Dados de Menores e Dados Sensíveis

Não coletamos dados de menores de 18 anos nem dados sensíveis (como saúde, religião ou orientação política).

7. Conformidade

Estamos em conformidade com a Política da Plataforma da Meta, com a LGPD (Brasil) e o GDPR (Europa), quando aplicável.

8. Alterações nesta Política

Podemos atualizar esta política periodicamente. A data da última atualização será sempre informada no topo.
